DocumentTools
=============
Coleção de ferramentas desenvolvidas em Python para processamento, extração e conversão de documentos.

Objetivo
========
Disponibilizar soluções simples e eficientes para automação de tarefas envolvendo documentos digitais.

Projetos
=========
1) PDF2Text Extractor
Ferramenta para extração de texto de arquivos PDF.

Recursos:
- Reconhece e extrai texto de PDFs nativos ou digitalizados
- Saída em DOCX
- Extração de páginas individuais ou múltiplas
- Estrutura preparada para integração com OCR

Repositório:
https://github.com/anselmosousa/PDF2Text-Extractor

2) Doc2Image Converter
Ferramenta para conversão de arquivos PDF e DOCX em imagens.

Recursos:
- Conversão de PDF para PNG/JPG
- Conversão de DOCX para PNG/JPG
- Exportação de páginas individuais ou múltiplas
- Configuração da resolução DPI da imagem de saída

Repositório:
https://github.com/anselmosousa/Doc2Image-Converter

Tecnologias
===========
- Python
- PyMuPDF
- Pillow
- OCR
- Processamento de documentos
